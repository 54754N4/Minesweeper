# Minesweeper
 Simple Java-based minesweeper 

## Keyboard Controls

#### Playing
> Reveal cell: SPACE

> Flag cell: CTRL + SPACE

> Reveal neighbours: SHIFT + SPACE

> Mark cell (type 1): Z

> Mark cell (type 2): X

#### Session
> New game: SHIFT + Z

> Set mines number: SHIFT + X

> Set board size: SHIFT + C

> Set cell size: SHIFT + V

> Print help: H

> Quit: ESCAPE
